## 2. 盒模型
css的一个基本概念是**盒模型**(box model)。

可见元素会在页面中占据一个矩形区域，该区域就是元素的盒子（box）,由四部分构成：**外边距、边框、内边距、元素内容**。

元素盒子有两个内容是可见的：**内容和边框**。
