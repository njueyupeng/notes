# 类型
## 七种数据类型

1. 空值（null）
2. 未定义（undefined）
3. 布尔值（boolean）
4. 数字（number）
5. 字符串（string）
6. 对象（object）
7. 符号（symbol ES6新增）

## typeof 

```javascript
 typeof undefined === 'undefined'

 typeof true === 'boolean'

 typeof 42 === 'number'

 typeof '42' === 'string'

 typeof {name:'name'} === 'object'

 typeof Symbol() === 'symbol'

 typeof null  === 'object'

```
