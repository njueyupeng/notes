# 模板语法

## 插值

### 文本
```html
<span>Messge:{{msg}}</span>
<span v-once>{{msg}}</span>
```
### 纯HTML
```html
<div v-html="rawhtml"></div>
```

### 特性
```html
<div v-bind:id="dynamicId"></div>
<button v-bind:disabled="idBUttonDisabled">Button</button>
```

### 使用JavaScript表达式
```html
{{bumber+1}}
{{ok?'yes':'no'}}
{{message.split('').reverse().join('')}}
<div v-bind:id="'list'+id"></div>
```
